    ### Die Konfigurationen:

    ```
        * PC0:
            - Gateway   :   192.168.1.1
            - Subnet    :   255.255.255.0
            - IP        :   192.168.1.10


        * PC1:
            - Gateway   :   192.168.2.1
            - Subnet    :   255.255.255.0
            - IP        :   192.168.2.10


        * PC2:
            - Gateway   :   192.168.3.1
            - Subnet    :   255.255.255.0
            - IP        :   192.168.3.10
    ```
    ### Einstellungen

    Einstellen per:
    1.  PC doppelklick
    2.  Tab: Desktop,
    3.  Oben links auf IP-Configuration
    4. Eingaben
        1. Gateway eingeben
        2. Subnetz eingeben
        3. IP eingeben
        4. Speichern
